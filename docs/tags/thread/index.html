    ThreadSafety

    August 8, 2016
    software, engineering
    java, thread

    A few days ago I ended up into this StackOverflow thread from almost seven years ago, this guy was getting inconsistent results from a multithread code. Well, looked like an excellent excuse to recall some topics and do some new research. To start with, let’s just quickly remember the differences between Multiprogramming, Multiprocessing, Multitasking, and Multithreading.
